The System Admin pages are used to administer system data and settings for the Niord application. It has been divided into the following sections:

Message Series
Manage the message series used for message series identifier.
Domains
Manage the domains.
Promulgation Types
Manage message promulgation types.
Firing Schedules
Manage the list of firing schedules.
Dictionaries
Manage the dictionaries.
Parameter Types
Manage the parameter types used for executing message templates.
Script Resources
Manage the script resources (Freemarker & JavaScript) used for message templates, print reports and mails.
Categories
Manage the list of categories used for grouping NW and NM messages and used for message templates.
Print Reports
Manage the PDF print reports.
Mail Queue
Search all pending and sent e-mails.
Niord Integration
Integration with other Niord servers.
Legacy Integration
Handle integration with the legacy systems.
Settings
Manage the system settings.
Batch Jobs
Monitor and manage batch jobs.
