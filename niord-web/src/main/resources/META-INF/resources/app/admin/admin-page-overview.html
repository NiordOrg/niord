The Admin pages are used to administer base data and settings for the Niord application. It has been divided into the following sections:

Charts
Configure the nautical charts.
Areas
Manage the list of areas used for grouping NW and NM messages.
Publications
Manage the list of publications that can be assigned to NW and NM messages.
Sources
Manage the list of sources that can be assigned to NW and NM messages.
Users & Contacts
Manage users and their group membership for the current domain, and manage contact used e.g. in mailing lists.
Mailing Lists
Manage mailing lists and the associated users and contacts.
